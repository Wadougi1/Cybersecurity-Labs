# SIEM Investigations

This module documents threat hunting and log investigation activities conducted using the **Elastic Stack (Kibana and Elasticsearch)** as part of graduate cybersecurity coursework and independent security analysis.

## Research Context

Security Information and Event Management (SIEM) platforms are the backbone of modern Security Operations Centers (SOCs). Analysts use SIEM tools to aggregate, search, and visualize log data from across an organization's infrastructure in real time.

This module demonstrates hands-on experience with:
- constructing targeted queries to isolate suspicious activity,
- interpreting structured log fields to identify threat indicators,
- and documenting findings in a format suitable for incident response or escalation.
